# Practice 2: Multiple Functions Operations

## Author
Luis Fernando Monjaraz Briseño

## Description
This C program demonstrates the implementation of multiple fundamental functions in C, including:

- **Sum of Arrays**: Calculates the sum of all elements in a numeric array
- **Power Calculation**: Computes base^exponent without using the pow() function
- **Character Counting**: Counts characters while ignoring spaces and line breaks
- **Vowels and Consonants Count**: Analyzes text to count vowels and consonants
- **Even/Odd Validation**: Determines if a number is even or odd
- **4-bit Binary Translator**: Converts numbers 0-15 to 4-bit binary representation

## Features
- Input validation for all functions
- Efficient buffer handling
- Clean and organized code structure
- Interactive user interface with loop control

## How to Compile and Run
```bash
gcc multiple_functions_operations.c -o multiple_functions_operations
./multiple_functions_operations
```

## Requirements
- GCC compiler or compatible C compiler
- Standard C library
